Auto Clicker
=================================

Isi:
- 1_SafeAutoClicker_AutoIt.au3
- macro_saver/ untuk menyimpan file macro .sacm

Cara jalan:
1. Install AutoIt.
2. Jalankan 1_SafeAutoClicker_AutoIt.au3.
3. Compile ke EXE jika sudah cocok.


File setting:
- safe_auto_clicker_settings.ini
- Disimpan di folder yang sama dengan script/exe.

Folder macro:
- macro_saver/
- Semua macro yang disimpan manual sebaiknya taruh di folder ini.
